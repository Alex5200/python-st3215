# Архитектура системы ST3215 Servo Controller

## Обзор системы

Система управления сервомоторами ST3215 состоит из нескольких взаимосвязанных модулей, обеспечивающих контроль до 6 сервоприводов через ROS2, GUI и программный интерфейс.

## 1. Общая архитектура системы

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              ОПЕРАТОР (USER)                                │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                    ┌─────────────────┼─────────────────┐
                    │                 │                 │
                    ▼                 ▼                 ▼
            ┌───────────────┐ ┌───────────────┐ ┌───────────────┐
            │   GUI PyQt6   │ │   ROS2 Node   │ │  Python API   │
            │  Controller   │ │   (rclpy)     │ │  (Library)    │
            └───────────────┘ └───────────────┘ └───────────────┘
                    │                 │                 │
                    └─────────────────┼─────────────────┘
                                      │
                                      ▼
                            ┌─────────────────┐
                            │   ST3215 Bus    │
                            │  (Serial/USB)   │
                            └─────────────────┘
                                      │
                    ┌─────────────────┼─────────────────┐
                    │                 │                 │
                    ▼                 ▼                 ▼
              ┌───────────┐     ┌───────────┐     ┌───────────┐
              │  Servo 1  │     │  Servo 2  │     │  Servo 6  │
              │ (Axis 1)  │     │ (Axis 2)  │     │ (Axis 6)  │
              └───────────┘     └───────────┘     └───────────┘
```

## 2. Модульная структура проекта

```
python-st3215/
├── src/python_st3215/          # Основное ядро библиотеки
│   ├── __init__.py             # Экспорт API
│   ├── st3215.py               # Контроллер ST3215 (основной класс)
│   ├── servo.py                # Класс Servo (обёртка для одного мотора)
│   ├── registers.py            # Реестры EEPROM и SRAM
│   ├── instructions.py         # Инструкции протокола
│   ├── errors.py               # Исключения
│   ├── decorators.py           # Декораторы валидации
│   └── ros2_node.py            # ROS2 узел + лимиты + валидация
│
├── gui/                        # GUI приложение
│   ├── __init__.py
│   ├── st3215_controller.py    # Основное GUI приложение
│   └── README.md
│
├── tests/                      # Тесты
│   ├── test_servo_driver_node.py

│
├── examples/                   # Примеры использования
├── main.py                     # Точка входа ROS2
└── pyproject.toml              # Конфигурация проекта
```

## 7. Взаимодействие оператор-система

### 7.1. Сценарии использования (Use Cases)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           ОПЕРАТОР (USER)                                   │
└─────────────────────────────────────────────────────────────────────────────┘
                │
                │ UC1: Подключиться к серво
                ├────────────────────────────────────────►
                │   1. Выбрать COM-порт
                │   2. Нажать "Connect"
                │   3. Проверить индикатор статуса
                │
                │ UC2: Двигать серво
                ├────────────────────────────────────────►
                │   1. Переместить слайдер оси
                │   2. Система проверяет лимиты
                │   3. Серво движется к позиции
                │
                │ UC3: Настроить лимиты
                ├────────────────────────────────────────►
                │   1. Нажать "Configure Limits"
                │   2. Задать min/max для каждой оси
                │   3. Включить/выключить проверку
                │   4. Сохранить
                │
                │ UC4: Создать программу
                ├────────────────────────────────────────►
                │   1. Добавить блоки команд
                │   2. Настроить параметры блоков
                │   3. Нажать "Run Program"
                │
                │ UC5: Тестировать границы
                ├────────────────────────────────────────►
                │   1. Выбрать ось
                │   2. Нажать "Test Boundaries"
                │   3. Серво движется: min → max → center
                │
                │ UC6: Сохранить/загрузить
                ├────────────────────────────────────────►
                    1. File → Save Program/Configuration
                    2. Выбрать файл
                    3. Подтвердить
```

## 9. Протокол обмена данными

### 9.1. Формат пакета ST3215

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           ИНСТРУКЦИЯ (WRITE)                                │
├─────────────────────────────────────────────────────────────────────────────┤
│  Header     ID    Length   Instruction   Parameters   Checksum             │
│  0xFF 0xFF  0x01  0x04     0x03          [data...]    ~sum                 │
│  (2 bytes)  (1)   (1)      (1)           (N bytes)    (1)                  │
└─────────────────────────────────────────────────────────────────────────────┘

                           ↓ Отправляется через Serial

┌─────────────────────────────────────────────────────────────────────────────┐
│                           ОТВЕТ (RESPONSE)                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│  Header     ID    Length   Error      Parameters   Checksum                │
│  0xFF 0xFF  0x01  0x03     0x00       [data...]    ~sum                    │
│  (2 bytes)  (1)   (1)      (1)         (N bytes)    (1)                    │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 9.2. Временная диаграмма обмена

```
Controller ────┬─────────────────────────────────────────────┬───────────────
               │                                             │
               │  [0xFF][0xFF][ID][LEN][INST][PARAM][CHK]   │
               │  ────────────────► SERVO                   │
               │                                             │
               │                                      Processing
               │                                             │
               │  [0xFF][0xFF][ID][LEN][ERR][PARAM][CHK]    │
               │  ◄──────────────── SERVO                   │
               │                                             │
               ▼                                             ▼
          T1 (запись)                                   T2 (ответ)

          T1 + T2 ≈ 2-4 ms (на 1Mbps)
```
